# Usage
<!-- usage -->
```sh-session
$ npm install -g flarum-cli
$ flarum-cli COMMAND
running command...
$ flarum-cli (-v|--version|version)
flarum-cli/1.0.0-alpha.2 linux-x64 node-v12.18.4
$ flarum-cli --help [COMMAND]
USAGE
  $ flarum-cli COMMAND
...
```
<!-- usagestop -->

To see a list of available commands, run:

```sh-session
flarum-cli
```

or

```sh-session
flarum-cli --help
```
